# Data Sources

- `workbooks.csv` - 28 workbook assets
- `extract_runs.csv` - 2,520 extract refresh records
- `performance_samples.csv` - 3,200 load-time and usage samples
- `support_tickets.csv` - 720 stakeholder issue records
- `metric_definitions.csv` - 64 KPI definition records
